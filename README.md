# HY248
一个基于HY中学24届8班的人际关系的项目，采用PCA、t-SNE、UMAP、NMF等算法。

## Run|运行
需要下载的库有：
```batch
pip install scikit-learn
pip install matplotlib
pip install numpy
pip install pandas
pip install umap-learn
```

接下来运行各个代码文件即可。
